# 🗄️ Projet Windows Active Disectory. Emulateur Windows 10 - UTM

> Auteurs : Evan Bonnal, Natalia Giraldo, Ludovic Dos Santos
> Formation : Bachelor IT en Cybersécurité
> Durée : 2 semaines
> Établissement : LaPlateforme_

---

## 📝 Description
Ce projet consiste en la conception, l'installation et la configuration complète d'un serveur Active Directory sous Windows 10 en emulateur UTM sur MacOS.

---

## 🛠️ Technologies Utilisées

- OS : Windows 10 - Emulateur UTM (ARM)
- Windows Active Director
- Réseau & Protocoles : Apache2, Samba, OpenSSH
- Administration : Powershell
